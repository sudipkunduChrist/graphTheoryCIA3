# 📍 Christ-Eats: Campus Food Navigator
**GRAPH THEORY:**(Unit 3: Graphs & Trees)  
**Campus:** CHRIST (Deemed to be University), Delhi NCR  

## 📖 Project Overview
This project applies **Graph Theory** to optimize student life. We modeled the Delhi NCR campus as a **Weighted Undirected Graph** where vertices are blocks/outlets and edges are walking paths.

## 🛠 Unit 3 Concepts Applied
* **Adjacency List:** Used to store campus connectivity.
* **Dijkstra's Algorithm:** Used for finding the Shortest Path from the user's location to the nearest outlet.
* **Tree Traversal:** Simulated for category-based searches (e.g., all outlets under the 'Cafeteria' parent node).
